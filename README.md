# Global Monitor

Viikoittainen uutiskatsaus 7 alueelta: Venäjä & Ukraina, Keski-Aasia & Kaukasus, Kiina, Muu Aasia, Intia, Lat. Amerikka, Afrikka.

Kolme näkymää: Uutiset, Talous, Demokratia (ml. kansalaisyhteiskunta).

## Rakenne

```
index.html    ← sovellus (avaa selaimessa)
weekly.json   ← viikon data (päivitä tämä)
```

## Käyttö

1. Avaa `index.html` selaimessa, tai
2. Julkaise GitHub Pagesissa (Settings → Pages → main branch)

## Päivitys

Korvaa `weekly.json` uudella versiolla → sivu päivittyy automaattisesti.
